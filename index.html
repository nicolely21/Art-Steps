Welcome to Art Steps!

Hello, you may have come to this website because you want to learn how to draw settings such as bedrooms, a shop or even a whole town. This website will give you a tutorial on how to draw a setting step by step. I will also give information on what brushes to use and how to operate on your drawing software.

What I use:

The software I use is photoshop, this is a software for laptops and computers. I also added extra brushes onto photoshop which you can browse through the adobe website. I suggest using a drawing tablet because without it, it will be very hard to draw. I use the Wacom cintiq 16 which is a drawing tablet that comes with a screen to project my work on. These can be pretty pricy so if you want to go for a cheaper one, you can get the drawing tablet that doesn't project the screen and instead, can only see it on your device.
